\section{Introduction}

\subsection{About}

The \pkg{xleading} package can be thought of as an extension to the
\pkg{leading} package even though it does not actually load the \pkg{leading}
package.
This package provides a single command \cs{leading} which sets the leading of
a paragraph---the baseline to baseline distance between lines---to a specified
length.

\subsection{Loading the package}

\listheading{Requirements:}
\begin{itemize}
  \item \LaTeXe{} version 2023-11-01 or newer
  \item \pkg{l3kernel} version 2023-11-01 or newer
  \item \pkg{xleading} is incompatible with the \pkg{leading} package.
\end{itemize}


\section{Command}

The syntax \meta{dimen} has the same meaning as the argument to \cs{dimeval},
which is documented in \pkg{usrguide}.

\begin{code}{\leading}
  \begin{syntax}
    \cs{leading} \marg{dimen}
  \end{syntax}
  Uses \cs{linespread}\marg{factor}\cs{selectfont} with a value of
  \meta{factor} set such that the new \cs{baselineskip} equals \meta{dimen}.
  The new \cs{baselineskip} has no stretch and shrink components.
  Cannot be used in math mode.
\end{code}


\section{Programming}

This package uses \cs{text_declare_purify_equivalent:Nn} to make
\cs{text_purify:n} correctly remove \cs{leading}.
